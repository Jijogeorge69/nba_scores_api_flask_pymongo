2018 NBA Scores REST API

By Daniel Lee

January 4, 2019

Description

This is a simple REST API that receives GET requests from the client and retrieves 2018 National Basketball Association (NBA) scores.

Data Scraping Implementation

The data is obtained through scraping the espn.com website using Python (v. 3.6.7), Scrapy (v. 1.5.1), and Splash (v. 0.7.2). The data was stored in a MongoDB database. The web scraping app is deployed on Heroku. You can find the code here.

REST API Implementation

The REST API was implemented using Python and Flask.
